# Тема 8 - Игра с системой авторизации и статистикой

## Описание
Полноценное игровое Windows Forms приложение с системой аутентификации пользователей, несколькими уровнями сложности, отслеживанием результатов и настройками.

## Функциональность
Комплексное приложение, включающее систему входа, игровую механику, сохранение результатов и статистику прохождения.

## Возможности

### Система аутентификации (LoginForm)
- **Регистрация пользователей**: Создание новых аккаунтов
- **Вход в систему**: Аутентификация по логину
- **Валидация**: Проверка уникальности логинов
- **Сессия**: Отслеживание текущего пользователя

### Игровой процесс (MainForm)
- **Несколько уровней сложности**:
  - Лёгкий (Easy)
  - Средний (Medium)
  - Сложный (Hard)
- **Игровая механика**: Зависит от выбранной сложности
- **Таймер**: Отслеживание времени прохождения
- **Результаты**: Автоматическое сохранение после игры

### Отслеживание результатов (ResultsForm)
- **История игр**: Просмотр всех прошлых результатов
- **Фильтрация**: По пользователю, дате, результату
- **Статистика**:
  - Количество побед/поражений
  - Среднее время прохождения
  - Любимая сложность
  - Процент успеха
- **Сортировка**: По различным параметрам

### Настройки (SettingsForm)
- **Настройки игры**: Параметры сложности
- **Графические настройки**: Цветовая схема, размеры
- **Звуковые эффекты**: Включение/отключение звуков
- **Профиль**: Информация о текущем пользователе
- **Сброс статистики**: Очистка результатов

### Сохранение данных (Storage)
- **Персистентность**: Автоматическое сохранение данных
- **Сериализация**: Бинарная сериализация объектов
- **Восстановление**: Загрузка данных при запуске
- **Безопасность**: Защита от потери данных

## Технические детали
- **Платформа**: .NET Framework 4.7.2
- **Технология**: Windows Forms
- **Хранение данных**: Бинарная сериализация
- **Архитектура**: Model-View паттерн

## Модели данных

### `GameResult`
Класс для хранения результата одной игры.

**Свойства:**
```csharp
[Serializable]
public class GameResult
{
    public string Login { get; set; }        // Логин игрока
    public DateTime Date { get; set; }       // Дата и время игры
    public bool IsWin { get; set; }          // Победа или поражение
    public int TimeSeconds { get; set; }     // Время прохождения (секунды)
    public int Difficulty { get; set; }      // Уровень сложности (0-2)
}
```

**Конструктор:**
```csharp
public GameResult(string login, bool isWin, int time, int difficulty)
{
    Login = login;
    Date = DateTime.Now;
    IsWin = isWin;
    TimeSeconds = time;
    Difficulty = difficulty;
}
```

**Уровни сложности:**
- `0` - Лёгкий (Easy)
- `1` - Средний (Medium)
- `2` - Сложный (Hard)

### `CurrentUser`
Статический класс для отслеживания текущего авторизованного пользователя.

**Свойства:**
```csharp
public static class CurrentUser
{
    public static string Login { get; set; }  // Логин текущего пользователя
}
```

**Использование:**
```csharp
// После входа
CurrentUser.Login = "username";

// Проверка авторизации
if (!string.IsNullOrEmpty(CurrentUser.Login))
{
    // Пользователь авторизован
}
```

## Классы форм

### `LoginForm`
Форма для входа в систему и регистрации новых пользователей.

**Функциональность:**
- Текстовое поле для ввода логина

**Методы:**

- `ValidateLogin(string login)` - Проверка корректности логина
- `UserExists(string login)` - Проверка существования пользователя

### `MainForm`
Главная форма с игровым процессом.

**Функциональность:**
- Меню выбора сложности

### `ResultsForm`
Форма отображения истории игр и статистики.

**Функциональность:**

  - Общее количество игр
  - Количество побед/поражений
  - Процент побед
  - Среднее время прохождения

**Столбцы таблицы:**
- Логин
- Дата и время

**Методы:**

**Разделы настроек:**

3. **Звуковые настройки**:

## Использование

### Первый запуск
1. Приложение откроет форму входа (LoginForm)
2. Нажать "Регистрация"
3. Ввести желаемый логин
4. Нажать "Зарегистрироваться"
5. Войти с новым логином

### Игра
1. После входа откроется главная форма (MainForm)
2. Выбрать уровень сложности:
   - **Лёгкий**: Больше времени, проще задания
   - **Средний**: Стандартные параметры
   - **Сложный**: Меньше времени, сложнее задания
3. Нажать "Начать игру"
4. Играть до победы или поражения
5. Результат автоматически сохранится

### Просмотр результатов
1. В главном меню выбрать "Результаты"
2. Откроется ResultsForm с таблицей всех игр
3. Использовать фильтры для поиска конкретных результатов
4. Просмотреть статистику в правой панели
5. При желании экспортировать данные

### Настройки
1. В главном меню выбрать "Настройки"
2. Изменить нужные параметры
3. Нажать "Сохранить" для применения
4. Или "Отмена" для выхода без сохранения

## Формат файла результатов
Бинарная сериализация с использованием BinaryFormatter:
```
[FileHeader: "GAMERESULTS"]
[Version: 1.0]
[Count: N]
[GameResult1]
[GameResult2]
...
[GameResultN]
```

```csharp
double winRate = (double)wins / totalGames * 100;
```

### Среднее время
```csharp
double avgTime = results.Average(r => r.TimeSeconds);
```

### Лучший результат
```csharp
var bestResult = results
    .Where(r => r.IsWin)
    .OrderBy(r => r.TimeSeconds)
```
