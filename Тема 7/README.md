# Тема 7 - Векторный графический редактор

## Описание
Полнофункциональный векторный графический редактор на Windows Forms с объектно-ориентированной архитектурой, поддержкой различных геометрических фигур и возможностью сохранения в собственном формате .vec.

## Функциональность
Профессиональный графический редактор с иерархией фигур, трансформациями, сериализацией и управлением историей операций.

## Возможности

### Рисование фигур
- **Круг** (Circle) - окружность заданного радиуса
- **Эллипс** (Ellipse) - овал с настраиваемыми осями
- **Квадрат** (Square) - правильный четырёхугольник
- **Прямоугольник** (Rectangle) - прямоугольник произвольных размеров
- **Ромб** (Rhombus) - четырёхугольник с равными сторонами

### Управление фигурами
- **Выделение**: Клик по фигуре для выбора
- **Перемещение**: Drag & Drop выделенной фигуры
- **Изменение размера**: Маркеры по углам и сторонам
- **Удаление**: Delete для удаления выбранной фигуры
- **Копирование**: Дублирование фигур
- **Многослойность**: Z-order управление (на передний/задний план)

### Трансформации
- **Поворот** (Rotation): Вращение на заданный угол
- **Зеркалирование** (Mirroring):
  - Горизонтальное отражение
  - Вертикальное отражение
- **Масштабирование**: Изменение размеров

### Стилизация (Stroke)
- **Цвет линии**: Выбор цвета обводки
- **Толщина линии**: Настройка ширины (1-10 пикселей)
- **Цвет заливки**: Выбор цвета заполнения
- **Прозрачность**: Настройка альфа-канала

### Работа с файлами
- **Сохранение**: Экспорт в формат .vec (бинарная сериализация)
- **Загрузка**: Импорт из формата .vec
- **Новый документ**: Очистка холста

### История операций (Undo/Redo)
- **Undo**: Отмена последней операции (Ctrl+Z)
- **Redo**: Повтор отменённой операции (Ctrl+Y)
- **Stack-based**: Использование стека для хранения истории

## Технические детали
- **Платформа**: .NET Framework 4.7.2
- **Технология**: Windows Forms
- **Графика**: GDI+ (System.Drawing, System.Drawing.Drawing2D)
- **Архитектура**: ООП с наследованием и полиморфизмом
- **Сериализация**: BinaryFormatter

## Архитектура классов

### Иерархия наследования
```
Figure (abstract)
├── Circle

├── Square
├── RectangleFigure
└── Rhombus
```

### `Figure` (Абстрактный базовый класс)
Базовый класс для всех геометрических фигур.

**Свойства:**
- `X, Y` - Координаты фигуры
- `Width, Height` - Размеры
- `Stroke` - Стиль обводки и заливки
- `IsSelected` - Флаг выделения
- `RotationAngle` - Угол поворота
- `MirrorHorizontal` - Горизонтальное зеркалирование
- `MirrorVertical` - Вертикальное зеркалирование

**Методы:**
- `Draw(Graphics g)` - Отрисовка фигуры
- `DrawFigure(Graphics g)` - Абстрактный метод отрисовки (переопределяется в наследниках)
- `DrawSelectionBorder(Graphics g)` - Отрисовка рамки выделения с маркерами
- `Move(int dx, int dy)` - Перемещение фигуры
- `MoveTo(int x, int y)` - Установка позиции
- `Contains(Point point)` - Проверка попадания точки в фигуру
- `Clone()` - Клонирование фигуры (глубокая копия)
- `Rotate(float angle)` - Поворот фигуры
- `Mirror(bool horizontal, bool vertical)` - Зеркалирование
- `GetTypeName()` - Абстрактный метод получения имени типа
- `GetBounds()` - Получение ограничивающего прямоугольника

**Особенности:**
- Атрибут `[Serializable]` для бинарной сериализации
- Использование Matrix для трансформаций
- Обратная трансформация для проверки попадания точки

### `Circle` : `Figure`
Реализация круга/окружности.

**Отрисовка:**
```csharp

{
    // Применение трансформаций
    // Рисование эллипса с равными осями
}
```

### `Ellipse` : `Figure`
Реализация эллипса (овала).

**Отрисовка:**
```csharp
protected override void DrawFigure(Graphics g)
{
    // Применение трансформаций
    // Рисование эллипса
}
```

### `Square` : `Figure`
Реализация квадрата.

**Особенность:**
- Ширина всегда равна высоте

### `RectangleFigure` : `Figure`
Реализация прямоугольника.

**Особенность:**
- Независимые ширина и высота

### `Rhombus` : `Figure`
Реализация ромба.

**Отрисовка:**
```csharp
protected override void DrawFigure(Graphics g)
{
    // Вычисление 4 вершин ромба
    // Рисование через GraphicsPath
}
```

### `Stroke`
Класс для хранения информации о стиле обводки и заливки.

**Свойства:**
- `Color LineColor` - Цвет линии
- `float LineWidth` - Толщина линии
- `Color FillColor` - Цвет заливки
- `bool Fill` - Флаг заливки

**Атрибут:**
- `[Serializable]` - Поддержка сериализации

### `Canvas`
Класс для управления коллекцией фигур на холсте.

**Методы:**
- `AddFigure(Figure fig)` - Добавить фигуру
- `RemoveFigure(Figure fig)` - Удалить фигуру
- `GetFigureAt(Point pt)` - Получить фигуру по координатам
- `SelectFigure(Figure fig)` - Выделить фигуру
- `DeselectAll()` - Снять все выделения
- `BringToFront(Figure fig)` - На передний план
- `SendToBack(Figure fig)` - На задний план
- `Clear()` - Очистить холст
- `DrawAll(Graphics g)` - Отрисовать все фигуры

### `StackMemory`
Класс для реализации функциональности Undo/Redo.

**Структура:**
- Stack для отменённых операций
- Stack для повторяемых операций

**Методы:**
- `Push(object state)` - Сохранить состояние
- `Undo()` - Отменить последнее действие
- `Redo()` - Повторить отменённое действие
- `CanUndo()` - Проверка возможности отмены
- `CanRedo()` - Проверка возможности повтора
- `Clear()` - Очистить историю

## Формат файла .vec
Бинарный формат с использованием BinaryFormatter:
```
[FileHeader]
[Version: 1.0]
[FigureCount: N]
[Figure1: Serialized]
[Figure2: Serialized]
...
[FigureN: Serialized]
```

## Использование

### Создание фигур
1. Выбрать тип фигуры из панели инструментов
2. Кликнуть на холсте для создания
3. Фигура появится с параметрами по умолчанию

### Редактирование фигур
1. **Выделение**: Кликнуть по фигуре
2. **Перемещение**: Зажать и перетащить
3. **Изменение размера**: Потянуть за маркер по углам/сторонам
4. **Удаление**: Выделить и нажать Delete

### Трансформации
1. Выделить фигуру
2. Выбрать операцию:
   - **Поворот**: Ввести угол и применить
   - **Зеркалирование**: Выбрать ось отражения
3. Применить изменения

### Стилизация
1. Выделить фигуру
2. Открыть панель стилей
3. Настроить:
   - Цвет обводки (ColorDialog)
   - Толщина линии (NumericUpDown)
   - Цвет заливки (ColorDialog)
   - Включить/выключить заливку (CheckBox)

### Сохранение и загрузка
1. **Сохранение**:
   - File → Save (Ctrl+S)
   - Выбрать имя файла (расширение .vec)
   - Все фигуры сохранятся в бинарном формате
2. **Загрузка**:
   - File → Open (Ctrl+O)
   - Выбрать файл .vec
   - Фигуры загрузятся на холст

### Undo/Redo
- **Отмена**: Edit → Undo (Ctrl+Z)
- **Повтор**: Edit → Redo (Ctrl+Y)
- История сохраняет до 50 последних операций

## Технологии и паттерны

### ООП принципы
- **Наследование**: Иерархия Figure
- **Полиморфизм**: Переопределение DrawFigure()
- **Инкапсуляция**: Приватные поля с публичными свойствами
- **Абстракция**: Абстрактный класс Figure

### Паттерны проектирования
- **Template Method**: Figure.Draw()
- **Memento**: StackMemory для Undo/Redo
- **Factory**: Создание фигур
- **Composite**: Canvas с коллекцией фигур

### Технологии GDI+
- **Matrix**: Аффинные трансформации
- **GraphicsPath**: Сложные фигуры (ромб)
- **LinearGradientBrush**: Градиенты
- **SmoothingMode**: Сглаживание
- **CompositingQuality**: Качество композиции

## Примеры кода

### Создание и добавление фигуры
```csharp
Circle circle = new Circle(100, 100, 50, 50);
circle.Stroke.FillColor = Color.Blue;
circle.Stroke.LineColor = Color.Black;
circle.Stroke.LineWidth = 2;
canvas.AddFigure(circle);
```

### Поворот фигуры
```csharp
Figure selected = canvas.GetSelectedFigure();
if (selected != null)
{
    selected.Rotate(45); // Поворот на 45 градусов
}
```

### Клонирование фигуры
```csharp
Figure clone = originalFigure.Clone();
clone.MoveTo(clone.X + 20, clone.Y + 20);
canvas.AddFigure(clone);
```
