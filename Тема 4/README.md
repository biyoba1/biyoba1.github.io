# Тема 4 - Трекер курса доллара

## Описание
Windows Forms приложение для отслеживания и визуализации динамики курса доллара по месяцам с построением графических диаграмм.

## Функциональность
Приложение позволяет вести учёт курса доллара за выбранный год с возможностью сохранения, загрузки данных и построения красочных диаграмм.

## Возможности

### Ввод данных
- **Год**: Текстовое поле для ввода года
- **Месяц**: Выпадающий список с месяцами (Январь-Декабрь)
- **Курс доллара**: Поле для ввода курса (должен быть положительным)
- **Автоматическое переключение**: После добавления данных автоматически переходит к следующему месяцу

### Валидация
- Проверка наличия всех обязательных полей
- Курс должен быть положительным числом
- Обработка ошибок ввода с информативными сообщениями

### Управление данными
- **Добавление записей**: Добавление курса для выбранного месяца
- **Сохранение в файл**: Экспорт данных в текстовый файл (.txt)
- **Загрузка из файла**: Импорт ранее сохранённых данных
- **Таблица данных**: Отображение всех введённых данных в DataGridView

### Визуализация
- **Диаграмма столбцов**: Красочная визуализация динамики курса
- **Градиентный фон**: Современный дизайн с LinearGradient
- **Сетка**: Горизонтальные линии сетки для удобства чтения
- **Подписи**: Значения курса над столбцами, названия месяцев снизу
- **Оси координат**: Подписанные оси X (Месяц) и Y (Курс в ₽)
- **Заголовок**: Динамический заголовок с указанием года
- **Адаптивность**: Автоматическая перерисовка при изменении размера окна

## Технические детали
- **Платформа**: .NET Framework 4.7.2
- **Технология**: Windows Forms
- **Графика**: GDI+ (System.Drawing.Drawing2D)
- **Хранение данных**: Массив на основе ArrayCurrencyData

## Классы

### `Form1`
Главная форма приложения с элементами управления и логикой визуализации.

#### Основные методы:
- `btnAdd_Click()` - Добавление данных о курсе
- `btnSave_Click()` - Сохранение данных в файл
- `btnLoad_Click()` - Загрузка данных из файла
- `btnBuildChart_Click()` - Построение диаграммы
- `pnlChart_Paint()` - Отрисовка диаграммы с градиентами
- `UpdateTable()` - Обновление таблицы данных
- `OnResize()` - Адаптация при изменении размера

### `ArrayCurrencyData`
Класс для управления коллекцией данных о курсе валюты.

#### Свойства:
- `Year` - Год для отслеживания
- `Count` - Количество записей
- `this[int index]` - Индексатор для доступа к данным

#### Методы:
- `Add(CurrencyData data)` - Добавить запись
- `SaveToFile(string path)` - Сохранить в файл
- `LoadFromFile(string path)` - Загрузить из файла

### `CurrencyData`
Модель данных для хранения информации о курсе.

#### Свойства:
- `Month` - Название месяца
- `Rate` - Курс доллара

## Визуализация диаграммы

### Параметры отрисовки
- **Отступы**: Левый (60px), Правый (20px), Верхний (50px), Нижний (60px)
- **Фон панели**: Градиент WhiteSmoke → LightGray
- **Фон диаграммы**: Белый с серой рамкой
- **Столбцы**: Градиент от светло-зелёного к тёмно-зелёному
- **Рамка столбцов**: Тёмно-зелёная
- **Ширина столбцов**: Адаптивная (15-60px)

### Элементы диаграммы
1. **Заголовок**: "Динамика курса доллара в [год] году"
2. **Сетка**: 5 горизонтальных линий (0%, 25%, 50%, 75%, 100%)
3. **Подписи значений**: Над каждым столбцом (формат: 00.00)
4. **Подписи месяцев**: Под каждым столбцом
5. **Оси координат**: С подписями "Курс (₽)" и "Месяц"

### Технологии визуализации
- **SmoothingMode.AntiAlias**: Сглаживание графики
- **LinearGradientBrush**: Градиентная заливка
- **DashStyle.Dot**: Пунктирная сетка
- **StringFormat**: Выравнивание текста

## Запуск приложения
1. Открыть решение `WindowsFormsApp1.sln` в Visual Studio
2. Собрать проект (Build → Build Solution)
3. Запустить приложение (F5 или Debug → Start Debugging)

## Использование

### Добавление данных
1. Введите год (например, 2024)
2. Выберите месяц из выпадающего списка
3. Введите курс доллара (например, 75.50)
4. Нажмите кнопку "Добавить"
5. Повторите для других месяцев

### Построение диаграммы
1. Введите данные хотя бы для одного месяца
2. Нажмите кнопку "Построить диаграмму"
3. Диаграмма отобразится на панели справа

### Сохранение и загрузка
1. **Сохранение**:
   - Нажмите кнопку "Сохранить"
   - Выберите место и имя файла
   - Данные сохранятся в текстовом формате
2. **Загрузка**:
   - Нажмите кнопку "Загрузить"
   - Выберите ранее сохранённый файл
   - Данные загрузятся в таблицу
