# Тема 3 - Анимация движущегося квадрата

## Описание
Windows Forms приложение на .NET 8.0 с анимацией движущегося квадрата, который меняет цвет при столкновении с границами формы.

## Функциональность
Интерактивная графическая анимация с следующими возможностями:
- Автоматическое движение квадрата по траектории
- Смена направления при достижении границ
- Автоматическая смена цвета при изменении направления
- Управление анимацией (старт/стоп)
- Настройка параметров анимации

## Возможности

### Анимация
- **Движение**: Квадрат движется по прямоугольной траектории
- **Направления**: 4 направления движения (0-3)
  - 0: Вправо
  - 1: Вниз
  - 2: Влево
  - 3: Вверх
- **Смена цвета**: При каждом изменении направления квадрат меняет цвет
- **Доступные цвета**: Красный → Синий → Зелёный → Оранжевый

### Управление
- **Кнопка "Запустить/Остановить"**: Управление анимацией
- **Кнопка "Настройки"**: Открывает форму настроек (Form2)
  - Выбор цвета квадрата
  - Настройка скорости анимации
- **Клавиатура**: Любая клавиша останавливает анимацию

### Адаптивность
- Автоматическая адаптация при изменении размера окна
- Квадрат остаётся в пределах видимой области
- Сохранение отступов от краёв (10 пикселей)

## Технические детали
- **Платформа**: .NET 8.0 (Windows)
- **Технология**: Windows Forms
- **Язык**: C#
- **Графика**: GDI+ (System.Drawing)

## Параметры анимации
- **Размер квадрата**: 40x40 пикселей
- **Скорость по умолчанию**: 5 пикселей за такт
- **Отступы от краёв**: 10 пикселей
- **Начальная позиция**: (10, 10)
- **Начальный цвет**: Красный
- **Таймер**: Управляется через Timer component

## Методы класса Form1

### `InitializeSquare()`
Инициализирует начальные параметры квадрата (позиция, цвет, направление).

### `timer1_Tick()`
Обработчик события таймера - вызывает движение и перерисовку.

### `MoveSquare()`
Реализует логику движения квадрата с учётом направления и границ.

### `ChangeColor()`
Меняет цвет квадрата в зависимости от текущего направления.

### `OnPaint()`
Отрисовывает квадрат с текущим цветом и чёрной рамкой.

### `button1_Click()`
Переключает состояние анимации (запуск/остановка).

### `button2_Click()`
Открывает форму настроек и применяет изменения.

### `Form1_KeyPress()`
Останавливает анимацию при нажатии любой клавиши.

### `Form1_Resize()`
Адаптирует позицию квадрата при изменении размера окна.

## Использование

### Базовое использование
1. Запустите приложение
2. Нажмите кнопку "Запустить" для начала анимации
3. Квадрат начнёт двигаться и менять цвет при отражениях
4. Нажмите кнопку "Остановить" для паузы
5. Нажмите любую клавишу для экстренной остановки

### Настройка параметров
1. Нажмите кнопку "Настройки"
2. Выберите цвет квадрата
3. Настройте скорость анимации
4. Нажмите OK для применения изменений

## Особенности реализации
- **DoubleBuffering**: Предотвращает мерцание анимации
- **KeyPreview**: Включён для обработки клавиатурных событий на уровне формы
- **Timer-based animation**: Плавная анимация через System.Windows.Forms.Timer
- **Margin calculation**: Учитывает отступы для корректного отображения
