# Тема 1 - Вычисление математических рядов

## Описание
Windows Forms приложение для вычисления логарифмических функций с использованием разложения в ряд Тейлора.

## Функциональность
Программа вычисляет значение функции **ln((x+1)/(x-1))** двумя способами:
1. Точное значение через встроенную функцию `Math.Log`
2. Приближенное значение через суммирование ряда с заданной точностью

### Математическая формула ряда
```
f(x) = Σ [2 / ((2n+1) * x^(2n+1))]
```
где n = 0, 1, 2, ...

## Возможности
- Ввод параметров:
  - `ε` (эпсилон) - точность вычислений (0 < ε < 1)
  - `x` - значение аргумента (x > 1)
- Валидация входных данных
- Вычисление с настраиваемой точностью
- Отображение результатов:
  - Точное значение функции
  - Сумма ряда
  - Количество членов ряда, необходимых для достижения заданной точности
- Защита от переполнения (максимум 1,000,000 итераций)

## Запуск приложения
1. Открыть решение `Task1.sln` в Visual Studio
2. Восстановить NuGet пакеты
3. Собрать проект (Build Solution)
4. Запустить приложение (F5)

## Использование
1. Введите значение точности (эпсилон) в первое поле
2. Введите значение x во второе поле (должно быть больше 1)
3. Нажмите кнопку "Вычислить"
4. Программа отобразит:
   - Точное значение ln((x+1)/(x-1))
   - Приближенное значение через сумму ряда
   - Количество итераций

## Примеры использования
- **Пример 1**: ε = 0.001, x = 2
- **Пример 2**: ε = 0.0001, x = 5
- **Пример 3**: ε = 0.00001, x = 10
