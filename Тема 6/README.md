# Тема 6 - Викторина "Города и памятники России"

## Описание
Интерактивное Windows Forms приложение-викторина о городах и памятниках России с системой управления вопросами и XML-хранилищем данных.

## Функциональность
Полнофункциональная викторина с административной панелью для управления вопросами и игровым режимом для пользователей.

## Возможности

### Режим игры (GameForm)
- **Вопросы с вариантами ответов**: 4 варианта ответа на каждый вопрос
- **Изображения**: Визуальное сопровождение вопросов
- **Система подсказок**: Подсказки для сложных вопросов
- **Подсчёт результатов**: Отслеживание правильных и неправильных ответов
- **Прогресс**: Индикатор прохождения викторины

### Административная панель (AdminForm)
- **Создание вопросов**: Добавление новых вопросов
- **Редактирование**: Изменение существующих вопросов
- **Удаление**: Удаление вопросов
- **Загрузка изображений**: Привязка изображений к вопросам
- **Управление подсказками**: Добавление текстовых подсказок
- **Настройка ответов**: Указание правильного ответа из 4 вариантов

### Главная форма (Form1)
- **Выбор режима**: Игра или Администрирование
- **Статистика**: Отображение количества доступных вопросов
- **Навигация**: Простой переход между режимами

## Технические детали
- **Платформа**: .NET Framework 4.7.2
- **Технология**: Windows Forms
- **Хранение данных**: XML файлы
- **Язык**: C#

## Классы и их назначение

### `Question`
Модель данных для представления вопроса викторины.

**Свойства:**
- `ImagePath` (string) - Путь к изображению
- `Hint` (string) - Текст подсказки
- `Text` (string) - Текст вопроса
- `Answers` (List\<string\>) - Список из 4 вариантов ответа
- `CorrectIndex` (int) - Индекс правильного ответа (0-3)

**Конструктор:**
```csharp
public Question()
{
    Answers = new List<string>();
}
```

### `XMLManager`
Класс для работы с XML-файлами (сохранение и загрузка вопросов).

**Методы:**
- `SaveQuestions(List<Question> questions, string path)` - Сохранение в XML
- `LoadQuestions(string path)` - Загрузка из XML
- `ValidateXML(string path)` - Валидация XML-файла

### `Form1`
Главная форма для выбора режима работы.

**Элементы управления:**
- Кнопка "Начать игру" → переход в GameForm
- Кнопка "Администрирование" → переход в AdminForm
- Метка статистики с количеством вопросов

### `GameForm`
Форма для прохождения викторины.

**Функциональность:**
- Отображение вопроса и изображения
- 4 кнопки для выбора ответа
- Кнопка "Показать подсказку"
- Переход к следующему вопросу
- Подсчёт результатов
- Отображение финального счёта

### `AdminForm`
Административная панель для управления вопросами.

**Возможности:**
- Список всех вопросов
- Форма добавления нового вопроса
- Редактирование выбранного вопроса
- Удаление вопросов
- Кнопка загрузки изображения
- Выбор правильного ответа (RadioButton)
- Сохранение изменений в XML

## Формат XML-файла
```xml
<?xml version="1.0" encoding="utf-8"?>
<Questions>
  <Question>
    <ImagePath>images/kremlin.jpg</ImagePath>
    <Hint>Находится в столице России</Hint>
    <Text>Какой памятник изображён на фотографии?</Text>
    <Answers>
      <Answer>Кремль</Answer>
      <Answer>Эрмитаж</Answer>
      <Answer>Петергоф</Answer>
      <Answer>Третьяковская галерея</Answer>
    </Answers>
    <CorrectIndex>0</CorrectIndex>
  </Question>
  <!-- Другие вопросы -->
</Questions>
```

## Запуск приложения
1. Открыть решение `Задание 6.slnx` в Visual Studio 2022
2. Собрать проект (Build → Build Solution)
3. Запустить приложение (F5)

## Использование

### Режим игры
1. На главной форме нажать "Начать игру"
2. Прочитать вопрос и рассмотреть изображение
3. Выбрать один из 4 вариантов ответа
4. При необходимости использовать подсказку
5. Нажать "Следующий вопрос"
6. По завершении увидеть результаты

### Режим администрирования
1. На главной форме нажать "Администрирование"
2. **Добавление вопроса**:
   - Заполнить поле "Текст вопроса"
   - Нажать "Загрузить изображение" и выбрать файл
   - Ввести 4 варианта ответа
   - Выбрать правильный ответ (RadioButton)
   - Добавить подсказку (опционально)
   - Нажать "Добавить вопрос"
3. **Редактирование**:
   - Выбрать вопрос из списка
   - Изменить необходимые поля
   - Нажать "Сохранить изменения"
4. **Удаление**:
   - Выбрать вопрос из списка
   - Нажать "Удалить вопрос"
5. Нажать "Сохранить в XML" для сохранения всех изменений
