# Тема 2 - Геометрические вычисления (Вариант 15)

## Описание
Windows Forms приложение для решения задач вычислительной геометрии: нахождение точек пересечения прямой с окружностью и двух окружностей.

## Функциональность
Программа предоставляет два режима работы:

### Режим 1: Пересечение прямой и окружности
Находит точки пересечения прямой, заданной двумя точками, с окружностью.

**Входные данные:**
- Координаты двух точек прямой: (x₁, y₁) и (x₂, y₂)
- Центр окружности: (cx, cy)
- Радиус окружности: r

**Результат:**
- 0 точек (прямая не пересекает окружность)
- 1 точка (прямая касается окружности)
- 2 точки (прямая пересекает окружность)

### Режим 2: Пересечение двух окружностей
Находит точки пересечения двух окружностей.

**Входные данные:**
- Первая окружность: центр (cx₁, cy₁), радиус r₁
- Вторая окружность: центр (cx₂, cy₂), радиус r₂

**Результат:**
- 0 точек (окружности не пересекаются)
- 1 точка (окружности касаются)
- 2 точки (окружности пересекаются)
- Ошибка (окружности совпадают)

## Математический алгоритм

### Пересечение прямой и окружности
Использует параметрическое уравнение прямой и решает квадратное уравнение:
```
At² + Bt + C = 0
где:
A = dx² + dy²
B = 2(fx·dx + fy·dy)
C = fx² + fy² - r²
```

### Пересечение окружностей
1. Вычисляет расстояние между центрами: d
2. Проверяет условия пересечения:
   - d > r₁ + r₂ → не пересекаются
   - d < |r₁ - r₂| → одна внутри другой
3. Находит точки через геометрические формулы

## Классы

### `Point`
Представляет точку на плоскости с координатами (X, Y).
- Свойства: `X`, `Y`
- Метод `ToString()` - форматированный вывод

### `GeometryHelper`
Статический класс с методами геометрических вычислений:
- `FindLineCircleIntersection()` - пересечение прямой и окружности
- `FindCircleCircleIntersection()` - пересечение двух окружностей
- `ValidateCoordinates()` - валидация входных данных

## Примеры использования

### Пример 1: Прямая пересекает окружность
```
Прямая: (0, 0) - (10, 0)
Окружность: центр (5, 0), радиус 3
Результат: 2 точки пересечения
```

### Пример 2: Две окружности касаются
```
Окружность 1: центр (0, 0), радиус 5
Окружность 2: центр (10, 0), радиус 5
Результат: 1 точка касания (5, 0)
```

### Пример 3: Окружности пересекаются
```
Окружность 1: центр (0, 0), радиус 5
Окружность 2: центр (4, 0), радиус 4
Результат: 2 точки пересечения
```

## Обработка ошибок
- Радиус должен быть положительным числом
- Точки прямой должны быть различными
- Координаты должны быть конечными числами (не NaN, не Infinity)
- Совпадающие окружности вызывают исключение
